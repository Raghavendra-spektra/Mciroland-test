# Scenario 1 — Environment Reporting and Inventory Collection

## Task 1: Create an Environment Reporting Utility

### Lab Overview

The Operations team requires a utility that can collect server environment information and generate reports for troubleshooting, inventory tracking, and operational audits.

### Assessment Objectives

1. Create a Bash script named **env-report.sh**.
2. Retrieve the following information using environment variables:
   * Current username
   * Hostname
   * Home directory
   * Current shell
   * Current working directory
3. Create the directory **/home/labuser/reports** if it does not already exist.
4. Generate a report named **/home/labuser/reports/EnvironmentReport.txt**.
5. Store all collected information within the report file.

### Evaluation Criteria

* The script must execute without errors.
* The report file must be created successfully.
* All five required pieces of information must be present in the report.

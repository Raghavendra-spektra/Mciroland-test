# Scenario 3 — Disk Utilization Monitoring and Alerting

## Task 3: Implement Disk Monitoring and Alerting

### Lab Overview

The Operations team requires a monitoring utility capable of identifying low disk space conditions and maintaining historical disk usage logs.

### Assessment Objectives

1. Create a Bash script named **disk-alert.sh**.
2. Monitor the root filesystem (**/**).
3. Calculate and display.
   - Total disk size
   - Available free space
   - Free space percentage
4. Generate the following alert when free space falls below 20%:
   ```text
   WARNING: Disk space below threshold.
   ```
5. Display a healthy status message when sufficient free space is available.
6. Generate a log file named **/home/labuser/reports/DiskUsageLog.txt**.

### Log Requirements

The log must contain:
   - Filesystem
   - Total Size (GB)
   - Free Space (GB)
   - Free Percentage
   - Status
   - Date and Time

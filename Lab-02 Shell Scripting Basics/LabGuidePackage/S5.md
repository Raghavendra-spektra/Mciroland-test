# Scenario 5 — Reusable System Administration Toolkit

## Task 5: Develop a Reusable System Administration Toolkit

### Lab Overview

The System Administration team requires reusable shell functions that can be incorporated into future automation and troubleshooting projects.

### Assessment Objectives

1. Create a Bash script named **system-tools.sh**
2. Create a function named
   ```bash
   get_system_info()
   ```
   * The function must display:
       - Hostname
       - Operating System
       - Current User
3. Create a function named
   ```bash
   get_disk_status()
   ```   
   * The function must display:
       - Filesystem
       - Total Space
       - Available Space
4. Call both functions within the script.
5. Display results on screen.
6. Save the combined output to:
   ```text
   / home/labuser/reports/SystemInfoReport.txt
   ```

### Evaluation Criteria

* Script executes without errors.
* Both required functions are created and called successfully.
* System and disk information are displayed correctly.
* Output is saved successfully to the report file.
* Validation must be Successful.

```text
/home/labuser/reports/SystemInfoReport.txt
```

# Scenario 4 — Automated Backup Operations

## Task 4: Implement an Automated Backup Solution

### Lab Overview

The Support team requires a repeatable backup process for protecting critical business data and maintaining backup records.

### Assessment Objectives

1. Create a Bash script named **backup-folder.sh**.
2. Backup data from:
```text
/home/labuser/LabData
```
3. Store backup archives in:
```text
/home/labuser/backups
```
4. Create timestamped backup archives using the following format:
```text
yyyyMMdd-HHmmss
```
5. Create compressed backup archives using tar and gzip.
6. Display a success message upon completion.
7. Generate a report named **/home/labuser/reports/BackupLog.txt**.

### Report Requirements

The report must contain:
  - Backup Time
  - Source Path
  - Backup File Name
  - Backup Size
> Note: Create at least three sample files inside `/home/labuser/LabData` before executing the backup script.

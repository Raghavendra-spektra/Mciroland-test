# Scenario 2 — Directory Permissions

This scenario covers Linux file system permissions. You will configure a shared project directory with special permission bits to support collaborative access and automatic group inheritance.

---

## Task 1: Configure a Shared Project Directory

> **Note:** Follow the specified paths, ownership, and permission values exactly to ensure validation works correctly.

1. Create the directory `/opt/devops-workspace`, set its ownership to `root:devops`, and apply permissions `1770` (sticky bit enabled).

2. Apply the **setgid** bit to the directory so new files automatically inherit the **devops** group.

3. Create a test file named `test-inherit.txt` inside the directory and verify it has inherited the **devops** group ownership.
